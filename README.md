# Home Energy Copilot – G-Assist Plug-in

## Overview
Home Energy Copilot enables on-device AI to control and optimize home energy usage through Project G-Assist.  
It connects with IFTTT or MQTT smart devices and allows natural language commands such as:
- “Optimize my energy usage”
- “Turn off the charger”
- “Start EV charging at low tariff hours”

## Installation
```bash
git clone https://github.com/yourusername/home-energy-copilot.git
cd home-energy-copilot
pip install -r requirements.txt
python plugin.py
```

## Configuration
Add your IFTTT key and device topics to `config.json`.
